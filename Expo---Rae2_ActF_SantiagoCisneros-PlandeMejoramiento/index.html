Formularios HTML

La forma de interactuar con tus usuarios en la web

¿Qué son los formularios HTML?

Los formularios HTML son una parte esencial de las páginas web que permiten a los usuarios introducir datos y enviarlos a un servidor para su procesamiento. Piensa en ellos como los puentes de comunicación entre los usuarios y los sitios web.

Desde buscar en Google hasta iniciar sesión en redes sociales o comprar en línea, todos utilizamos formularios constantemente al navegar por internet.

Elementos básicos de un formulario

  • <form>: El contenedor principal
  • <input>: Para textos, contraseñas, etc.
  • <textarea>: Para textos largos
  • <select>: Listas desplegables
  • <button>: Botones de envío o acción

Atributos importantes

  • action: ¿A dónde se envían los datos?
  • method: Cómo se envían (GET o POST)
  • name: Identifica cada campo
  • type: Tipo de campo (text, email, password)
  • placeholder: Texto de ejemplo en un campo

Tipos de input comunes

  • text - Texto simple
  • password - Contraseñas
  • email - Dirección de correo
  • number - Números
  • checkbox - Casillas de verificación
  • radio - Botones de opción

Ejemplo de código HTML

<form action="/procesar.php" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" required>
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje"></textarea>
  
  <button type="submit">Enviar</button>
</form>
